El sistema "PORTAFOLIO".

Un portafolio digital se refiere a una recopilación de documentos que pueden mostrar diferentes aspectos globales o parciales de una persona (personales, académicos, profesionales) o de una organización (objetivos, organigrama, productos) los cuales han estado seleccionados y organizados de forma reflexiva y deliberada y son presentados en formato digital (texto, imágenes, animaciones, simulaciones, audio y vídeo). Se puede decir que es una manera de presentar una identidad en formato digital..

"Test conocimientos".

Esta aplicación, realiza una series de pregúntas de diferentes aspecto; tanto Geografico, Hístorico, Desporte, Musica, Entre otros. Donde el usuario debera responder y escalar hasta completar las rondas realizadas por el sistema, donde obtendra su resultado de las respuesta correctas e incorrectas y su puntuación, así mismo un raking del top de todos los jugadores.

Este Sistema se realizó bajo el lenguaje de programación PHP y Javascript y las herramientas: CSS, HTML, bootstrap. bajos los roles, tanto como Adminitrador y de Usuario, donde el admin puede agregar nuevas preguntas y puede ver que usuarios estan realizando las rondas de pregunta y sus estados y raking top.

Una pequeña instrucción sobre la aplicación

Una página web, que se carga a las ventas de los productores de diferentes sectores (CORRUPTO, AGROINDUSTRIA, RECICLAJE Y MEDIO AMBIENTE, EMBALAJE INDUSTRIAL, TRANSPORTE), lo que permite a los usuarios tener una mejor búsqueda de los productos y equipos que necesitan, para que puedan ver la calidad de los productos que la web les ofrece de diferentes empresas reconocidas en el país. Cuenta con registro para los usuarios, información detallada con una sección de noticias, también tiene detalles de cada producto.

Esta aplicación fue creada de forma dinámica para que se puedan añadir nuevos productos y nuevos cambios en la página web desde el administrador.
